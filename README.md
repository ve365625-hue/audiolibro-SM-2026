# audiolibro-SM-2026

Proyecto audiolibro de la clase de Sistemas Multimedia de la UAEH, ciclo escolar enero - junio 2026

Cada alumno debe:

1. Crear un fork del repositorio.
2. Agregar su carpeta personal.
3. Subir el contenido de su proyecto: arcivos .html, .js; carpetas images y sounds. El archivo .html debe renombrarse como index.html
4. Activar pages de github.
5. Realizar un Pull Request.

Nombre obligatorio de la carpeta:

ApellidosNombre/
